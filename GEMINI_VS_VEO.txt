┌─────────────────────────────────────────────────────────────────────────┐
│                      COMMON MISTAKES                                    │
└─────────────────────────────────────────────────────────────────────────┘

❌ Using generativelanguage.googleapis.com for Veo
   ➜ Use aiplatform.googleapis.com instead

❌ Using API key for Veo authentication
   ➜ Use Service Account with OAuth 2.0

❌ Expecting immediate video response
   ➜ Poll the long-running operation

❌ Using Gemini request format for Veo
   ➜ Use Vertex AI format with instances/parameters

❌ Forgetting to enable Vertex AI API
   ➜ Enable in GCP Console

❌ Missing IAM permissions
   ➜ Grant roles/aiplatform.user to service account

┌─────────────────────────────────────────────────────────────────────────┐
│                      QUICK SETUP GUIDE                                  │
└─────────────────────────────────────────────────────────────────────────┘

For GEMINI:
  1. Get API key from https://makersuite.google.com/app/apikey
  2. Add to .env: GEMINI_API_KEY=AIzaSy...
  3. Done! ✓

For VEO:
  1. Create GCP project
  2. Enable Vertex AI API
  3. Create service account
  4. Grant role: Vertex AI User
  5. Download JSON key
  6. Add to .env:
     GCP_PROJECT_ID=your-project
     GCP_LOCATION=us-central1
     GOOGLE_APPLICATION_CREDENTIALS=./config/service-account-key.json
  7. Store key file securely
  8. Test with: node src/geminiService.js

┌─────────────────────────────────────────────────────────────────────────┐
│                      WHEN TO USE WHICH?                                 │
└─────────────────────────────────────────────────────────────────────────┘

Use GEMINI when:
  ✓ Generating text/scripts
  ✓ Chatbots and Q&A
  ✓ Content summarization
  ✓ Simple image generation (Imagen)
  ✓ Quick prototyping
  ✓ Low budget projects

Use VEO when:
  ✓ Generating videos
  ✓ You have GCP infrastructure
  ✓ Need production-grade video generation
  ✓ Have budget for video AI
  ✓ Building video-centric apps

Can combine both:
  1. Use Gemini to generate video script
  2. Use Veo to generate video from script
  3. Use Gemini to generate video description
